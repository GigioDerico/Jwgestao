# JW Gestao - Guia de Uso para Treinamento

## 1. Objetivo deste guia

Este documento explica como usar o **JW Gestao** de forma simples e pratica. Ele foi escrito para treinamento de usuarios que vao trabalhar com as telas principais do sistema no dia a dia.

O foco deste guia e mostrar, em linguagem direta:

- como entrar no sistema
- como navegar pelas telas
- como cadastrar e editar informacoes
- como montar e manter escalas
- como exportar materiais
- como acompanhar notificacoes e confirmacoes

---

## 2. Antes de comecar

Para usar o sistema, o usuario precisa:

- ter acesso liberado no sistema
- estar vinculado a um membro cadastrado
- possuir telefone e senha validos para login

Dependendo do perfil, algumas telas podem nao aparecer ou podem abrir somente para consulta.

Perfis mais comuns:

- `coordenador`
- `secretario`
- `designador`
- `publicador`

O que cada usuario pode fazer depende das permissoes configuradas no sistema.

---

## 3. Como entrar no sistema

### Passo a passo do login

1. Abra o aplicativo.
2. Na tela inicial, informe o **telefone** cadastrado.
3. Informe a **senha**.
4. Clique ou toque em **Entrar**.

### O que acontece depois

- Se os dados estiverem corretos, o sistema abre a area principal.
- Se ja existir uma sessao valida, o sistema pode restaurar o acesso automaticamente.
- Se houver erro de login, revise telefone e senha.

Observacao:
O usuario entra com o telefone, mas o sistema trata esse dado internamente para fazer a autenticacao.

---

## 4. Como a navegacao funciona

Depois do login, o usuario vera:

- **menu lateral**
- **barra superior**
- **area principal da tela**

### Menu lateral

As telas principais sao:

- `Dashboard`
- `Membros`
- `Reunioes`
- `Designacoes`
- `Configuracoes`

### Submenu de Designacoes

O menu **Designacoes** e separado por tipo:

- `Reunioes`
- `Audio e Video`
- `Saida de Campo`
- `Carrinho`

Cada item abre uma pagina propria, com suas funcoes especificas.

### Barra superior

Na barra superior, o usuario encontra:

- acesso rapido as notificacoes
- atalhos rapidos da tela atual
- acesso ao perfil

---

## 5. Dashboard

O **Dashboard** e a primeira tela de trabalho depois do login.

### O que voce encontra no Dashboard

- indicadores rapidos do sistema
- resumo de membros e reunioes
- atalhos para areas mais usadas
- secao **Minhas Designacoes**

### Como usar

Use o dashboard para:

- conferir rapidamente o que esta acontecendo no sistema
- abrir modulos com um clique
- ver as proximas designacoes pessoais
- confirmar recebimento de designacoes quando essa acao estiver disponivel

### Minhas Designacoes

Essa secao mostra as designacoes ligadas ao membro que esta logado.

Nela, o usuario pode:

- ver a data da designacao
- identificar o tipo de atividade
- acompanhar o status
- confirmar recebimento quando exigido

---

## 6. Membros

O modulo **Membros** e usado para manter o cadastro dos irmaos e estudantes.

### O que pode ser feito

- listar membros
- pesquisar por nome
- filtrar por situacao espiritual
- filtrar por grupo de campo
- alternar a forma de visualizacao
- cadastrar novo membro
- editar membro existente

### Modos de visualizacao

O sistema pode mostrar os membros em:

- lista geral
- agrupamento por grupo de campo
- agrupamento por familia

Esses modos ajudam a localizar pessoas com mais facilidade.

### Como cadastrar um novo membro

1. Abra a tela `Membros`.
2. Clique ou toque em **Novo Membro**.
3. Preencha os dados principais.
4. Preencha os dados complementares, se necessario.
5. Revise as informacoes.
6. Salve o cadastro.

### Dados principais normalmente usados

- nome completo
- telefone
- email
- situacao espiritual
- genero
- grupo de campo
- organizacao familiar
- privilegios internos
- perfil de sistema
- contato de emergencia

### Endereco no cadastro e na edicao

O formulario atual tambem permite informar endereco estruturado:

- rua/logradouro
- numero
- bairro
- cidade
- estado (UF)
- CEP

Pontos importantes:

- o endereco e opcional
- a UF e tratada em maiusculas
- o CEP pode ser digitado com mascara visual
- o sistema armazena o CEP sem a mascara

### Como editar um membro

1. Localize o membro na lista.
2. Abra a acao de edicao.
3. Altere os dados necessarios.
4. Salve as mudancas.

Use a edicao sempre que precisar corrigir ou atualizar dados de contato, situacao espiritual, familia, grupo ou endereco.

---

## 7. Reunioes

A tela **Reunioes** e usada para consultar reunioes e gerar materiais para compartilhar ou imprimir.

### Tipos de reuniao

O sistema organiza as reunioes em dois grupos:

- `Meio de semana`
- `Fim de semana`

### O que pode ser feito

- consultar reunioes registradas
- alternar entre os tipos de reuniao
- ver responsaveis e partes
- gerar material para visualizacao e impressao
- exportar em imagem ou PDF

### Quando usar esta tela

Use esta tela quando a necessidade for:

- consultar o quadro da reuniao
- revisar a estrutura antes da impressao
- gerar um arquivo para compartilhar

Se a ideia for montar ou alterar as designacoes da reuniao, o caminho principal e **Designacoes > Reunioes**.

---

## 8. Designacoes de Reunioes

Essa area concentra a montagem das designacoes ligadas as reunioes congregacionais.

### O que pode ser feito

- criar registros de reuniao
- editar registros existentes
- definir horarios
- informar canticos
- preencher partes do meio de semana
- preencher partes do fim de semana
- ajustar designados por funcao

### Como usar na pratica

1. Abra `Designacoes > Reunioes`.
2. Escolha a reuniao ou o periodo desejado.
3. Preencha os campos da reuniao.
4. Defina os responsaveis por cada parte.
5. Confira se todas as partes foram preenchidas.
6. Revise a distribuicao com calma.
7. Salve as alteracoes.

### Quando revisar com atencao

Antes de salvar, confira:

- se os designados estao corretos
- se os horarios batem com a reuniao real
- se os canticos e titulos estao corretos
- se nao ha omissoes em partes obrigatorias

---

## 9. Designacoes de Audio e Video

Essa tela e usada para montar a escala tecnica das reunioes.

### Funcoes trabalhadas

Entre as funcoes disponiveis, o sistema pode organizar:

- som
- imagem
- palco
- microfone 1
- microfone 2
- entradas/auditorio

### Como montar a escala do mes

1. Abra `Designacoes > Audio e Video`.
2. Escolha o mes desejado.
3. Se a estrutura do mes ainda nao existir, use a acao de gerar o mes.
4. Revise as datas criadas.
5. Escolha os irmaos em cada funcao.
6. Revise os nomes.
7. Salve os ajustes necessarios.

### Comportamento automatico

- Quando o mes ja esta completamente gerado, a area de geracao mensal deixa de aparecer.
- O sistema ajuda a evitar repeticao indevida da mesma pessoa em funcoes restritas no mesmo dia.

### Exportacao

Ao final, voce pode exportar a escala em:

- `JPG`
- `PDF`

Isso e util para compartilhar, imprimir ou guardar a escala.

---

## 10. Designacoes de Saida de Campo

Essa tela organiza a escala mensal de saida de campo.

### Categorias usadas atualmente

- Terca-feira
- Quarta-feira
- Sexta-feira
- Sabado
- Sabado - Rural
- Domingo

### Como gerar e preencher a escala

1. Abra `Designacoes > Saida de Campo`.
2. Selecione o mes.
3. Gere a estrutura do mes, se ela ainda nao existir.
4. Preencha responsavel, horario e local em cada linha.
5. Adicione linhas extras quando necessario.
6. Revise o quadro completo.
7. Salve os ajustes, se houver.

### Linhas extras

Dependendo da necessidade, e possivel:

- adicionar grupos extras no domingo
- adicionar linhas de sabado rural

### Comportamento automatico

- Quando o mes ja esta completamente estruturado, a area de geracao do mes e ocultada.

### Exportacao

O quadro pode ser exportado em:

- `JPG`
- `PDF`

O material foi ajustado para tentar caber completo na exportacao, incluindo o quadro final de texto.

---

## 11. Designacoes de Carrinho

Essa tela controla a escala mensal do carrinho.

### O que pode ser feito

- criar linhas da escala manualmente
- separar por semana
- definir dia e dia da semana
- definir horario
- definir local
- definir publicador 1
- definir publicador 2
- editar linhas existentes

### Como montar a escala

1. Abra `Designacoes > Carrinho`.
2. Escolha o mes desejado.
3. Cadastre as linhas da escala.
4. Informe local, horario e publicadores.
5. Revise a distribuicao por semana.
6. Ajuste os registros quando necessario.
7. Revise antes de exportar.

### Visualizacao

- No desktop, a escala aparece em formato de tabela.
- No mobile, a escala aparece em cards.

### Legibilidade

As linhas possuem divisorias visuais para facilitar a leitura, principalmente para quem tem mais dificuldade para enxergar os registros com rapidez.

### Exportacao

A escala pode ser exportada em:

- `JPG`
- `PDF`

- criar grupos
- editar grupos
- remover grupos
- definir dirigente

Mantenha os grupos atualizados para facilitar filtros, organizacao e distribuicao nas outras telas.

## 15. Como exportar e imprimir

Algumas telas permitem gerar materiais prontos para compartilhar ou imprimir.

### Formatos disponiveis

- `JPG`
- `PDF`

### Quando usar JPG

Use `JPG` quando quiser:

- enviar em grupos de mensagem
- salvar como imagem
- compartilhar rapidamente

### Quando usar PDF

Use `PDF` quando quiser:

- imprimir
- arquivar
- manter um formato mais estavel para distribuicao

### Dica pratica

Antes de exportar:

- revise o mes correto
- confirme os nomes
- confirme horarios
- confirme locais

Isso ajuda a evitar retrabalho e reenvio de material corrigido depois.

## 16. Boas praticas de uso

Para usar o sistema com menos erros no dia a dia:

- atualize os membros antes de montar escalas novas
- revise grupos de saida de campo periodicamente
- confirme se o mes correto esta selecionado antes de editar
- sempre revise antes de exportar
- incentive os membros a confirmar designacoes recebidas
- use as configuracoes de permissao com cuidado, principalmente em perfis administrativos

---

## 17. Solucao de problemas comuns

### Nao consigo ver uma tela no menu

Possiveis causas:

- o perfil do usuario nao tem permissao
- a permissao foi alterada nas configuracoes

Verifique a matriz de acesso em `Configuracoes`.

### Nao consigo editar um registro

Possiveis causas:

- o usuario esta em perfil sem permissao de edicao
- a tela esta acessivel apenas para consulta naquele contexto

### A exportacao saiu errada

Antes de repetir:

- revise se o conteudo estava totalmente carregado
- confirme se o mes selecionado era o correto
- tente novamente no formato `JPG` ou `PDF`, conforme a necessidade

### Uma designacao nao aparece para o membro

Verifique:

- se a designacao foi realmente salva
- se ela foi vinculada ao membro correto
- se a notificacao correspondente foi gerada

3. atualizar membros, se necessario
4. montar ou ajustar designacoes
5. revisar escalas
6. exportar o material final
7. acompanhar confirmacoes dos membros

Esse fluxo cobre a maior parte do uso normal do sistema.
